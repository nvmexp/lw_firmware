Profile(s) Owner: GFW team

GFW has several profiles named gfw-chip-[engine]-profile_name

Profiles:

* Devinit
Profile for devinit firmware running under MP-SK on PMU/SOE. S mode only.

* FSP-FW
Profile for main FSP firmware running under MP-SK. S mode only.

* sec-fw
Profile for HULK firmware running on SEC2 under MP-SK. S mode only.
